## (Desafio Dio) Desenvolvendo seu blog com as tecnologias de Java e Angular

### Objetivos

Backend, Frontend, Git/Github.

### Requisitos

Lógica de Programação, HTML5/CSS3, JavaScript.

### Package.json

- npm init -y
- npm i json-server

### Angular

- Single-Page Application
- Componentes
- Angular CLI (https://cli.angular.io/)

### Instalação Angular

- npm i -g @angular/cli
- ng new blog
- cd blog
- ng serve
